CodeRunner

Collection of bookmarklets that lets you run code written in Etherpad with just 1-click!

We absolutely love collaborative code editing in Etherpad. The following bookmarklets take it one step ahead & allow you to run code written in Etherpad with just 1-click! It feels like hitting the 'Run' command in your IDE. You can start using these bookmarklets right away by just dragging them to your bookmarks bar or by right clicking the link & bookmarking it
