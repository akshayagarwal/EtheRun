CodeRunner

Collection of bookmarklets that lets you run code written in Etherpad with just 1-click!

Download this project as a .zip file Download this project as a tar.gz file

All of us love collaborative code editing in Etherpad but the only thing lacking is a code runner. The following bookmarklets allow you to run code written in Etherpad with just 1-click! It automatically copies your code from Etherpad & runs it in Codepad providing you the feel of hitting the 'Run' command in your IDE! You can start using these bookmarklets right away by just dragging them to your bookmarks bar or by right clicking the link & bookmarking it
